기본 슬라이더 (자동 재생)

  • 슬라이드 1

    무한 루프와 자동 재생 기능을 갖춘 모던 바닐라 JavaScript 캐러셀 슬라이더

  • 슬라이드 2

    CSS Transform을 활용한 GPU 가속 애니메이션으로 부드러운 전환 효과

  • 슬라이드 3

    터치 스와이프 지원으로 모바일에서도 편리하게 사용

  • 슬라이드 4

    접근성을 고려한 ARIA 속성 및 키보드 네비게이션 지원

설정: autoplay: true, delay: 3000ms, speed: 500ms

수동 제어 슬라이더

  • 수동 제어 모드

    자동 재생이 비활성화된 슬라이더입니다

  • 버튼으로 제어

    이전/다음 버튼이나 하단의 컨트롤을 사용하세요

  • 스와이프 지원

    모바일에서는 좌우로 스와이프하여 이동할 수 있습니다

설정: autoplay: false, speed: 600ms
